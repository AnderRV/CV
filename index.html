Desarrollo de la web y el backoffice del sitio de reservas de restaurantes Restalo

  • Restalo es el portal referencia de reservas online. La web está desarrollada con el framework de PHP Yii. Basada en un modelo MVC clásico, con uso de AMQP para tareas asíncronas como logs, Solr para búsquedas, Memcached como sistema de caché y Mandrill para envío de mails es una aplicación con diversas partes bien separadas. También tiene un backoffice para la gestión interna.

Creación de la aplicación web de Egolike mientras desarrollaban en paralelo la de iOS

  • Egolike es una aplicación social. Basada en una API desarrollada con Java en Google App Engine, la web es una aplicación PHP con WordPress y Silex con un uso intensivo de Javascript en el front-end.

Desarrollador con Symfony y NodeJs, tanto front-end como back-end

  • Trabajamos en el proyecto Battle for Gea. Es un juego de navegador de tipo CGG (juegos de cartas coleccionables) basado en un mundo de fantasía. La parte web funciona sobre Symfony2 con Doctrine, Twig y MySQL. La parte del juego funciona con una conexión vía socket.io a un proyecto NodeJS con MongoDB. El cliente del juego está totalmente en JavaScript con un uso intensivo de CSS3, por ejemplo para las animaciones.

Varios proyectos paralelos, tanto internos como externos

  • Alianzo es el principal proyecto. Basado en Symfony2 con Doctrine como ORM, Twig como sistema de plantillas y MySQL. Está centrado en rankear personas y marcas en los medios sociales como Facebook, Twitter y GooglePlus entre otros. Para ellos se hace uso de las APIs en un rastreo periódico de datos y después se computan con un algoritmo interno.
  • Metroo es una red social local. Ha sido migrada a WordPress después, pero estaba basada en Symfony1.4 con Propel como ORM y MySQL. Tiene una clara diferenciación entre la propia red social y los Puntos de Interés, como pueden ser negocios, playas, monumentos o eventos. La parte social se centra mucho en la zona del usuario para mostrarle información relevante. Por el otro lado, la parte de negocios está basada en el rastreo de puntos a insertar y comentarios de usuarios, tanto internos (de los propios miembros el la red) como obtenidos de páginas externas.
  • Otros proyectos de menos envegadura como Gananzia o Startup2 basados en WordPress, Aupatu un “clon” de Digg o Menéame, Ibotanika una red social orientada a plantas y naturaleza que ya no está activa, Mapastur (como parte de mapalia.com que luego acabaría siendo Metroo) es un sitio basado en puntos de interés y localización.
